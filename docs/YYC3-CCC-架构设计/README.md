
## 🔧 技术栈

### 前端技术

- **框架**：Next.js 15.0.3 + React 19
- **语言**：TypeScript 5.0.0
- **样式**：Tailwind CSS 3.4.17
- **组件库**：shadcn/ui + Radix UI
- **动画**：Framer Motion 12.23.12
- **3D 场景**：@splinetool/react-spline 4.1.0
- **粒子效果**：@tsparticles/react 3.0.0

### 后端技术

- **API**：Next.js API Routes
- **数据库**：PostgreSQL 15
- **缓存**：Redis 7
- **认证**：JWT
- **监控**：Sentry

### 开发工具

- **包管理**：npm
- **代码检查**：ESLint
- **格式化**：Prettier
- **测试**：Vitest 2.1.5
- **版本控制**：Git
- **CI/CD**：GitHub Actions

## 📝 架构原则

### 五高原则

- **高可用性**：确保系统 7x24 小时稳定运行
- **高性能**：优化加载速度和交互响应
- **高安全性**：保护用户数据和隐私安全
- **高扩展性**：支持业务快速扩展
- **高可维护性**：便于后续维护和升级

### 五标体系

- **标准化**：统一的技术和流程标准
- **规范化**：严格的开发和管理规范
- **自动化**：提高开发效率和质量
- **智能化**：利用自动化工具提升能力
- **可视化**：直观的监控和管理界面

### 五化架构

- **流程化**：标准化的开发流程
- **文档化**：完善的文档体系
- **工具化**：高效的开发工具链
- **数字化**：数据驱动的决策
- **生态化**：开放的生态系统
