```
components/
├── ui/                 # UI 基础组件
│   ├── button.tsx      # 按钮组件
│   ├── card.tsx        # 卡片组件
│   └── input.tsx       # 输入框组件
├── features/           # 功能组件
│   ├── auth/          # 认证功能
│   ├── services/       # 服务功能
│   └── pricing/       # 定价功能
├── layouts/           # 布局组件
│   ├── header.tsx     # 头部布局
│   ├── footer.tsx     # 底部布局
│   └── sidebar.tsx    # 侧边栏布局
└── common/            # 通用组件
    ├── loading.tsx    # 加载组件
    └── error.tsx      # 错误组件
```

## 📝 设计规范

### UI 设计原则

- **一致性**：保持界面风格一致
- **简洁性**：界面简洁明了
- **易用性**：操作简单直观
- **美观性**：视觉效果美观

### UX 设计原则

- **用户中心**：以用户为中心
- **易学性**：易于学习和使用
- **效率性**：提高操作效率
- **容错性**：容错和恢复
